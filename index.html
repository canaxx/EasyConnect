EasyConnect

A tabbed remote desktop application with a Chrome-like UI and support for RDP, VNC, SSH, Telnet, and remote PowerShell connections.

Fork me on GitHub


Current version: 3.1.0.102

Efficient

Manage all of your remote connections from within a single tabbed interface with support for tab tearing to new windows and tab reorganization.

Extensible

You can extend EasyConnect by writing plugins for other remote desktop protocols. Plugins for RDP, VNC, SSH, Telnet, and PowerShell are included out of the box.

Secure

Your saved passwords are encrypted with an RSA key stored in your machine's key store.

Installation Options

Available to install through the Microsoft Store, WinGet, and Chocolatey.

Automatic updates

Automatic updating with the Microsoft Store or NetSparkle keeps you running on the latest version.

Open source

EasyConnect is open source software and is completely free for commercial use. It is licensed under the GPL.
